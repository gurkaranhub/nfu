*Description:*

Returns every $$n$$th row, starting with row 1. (More precisely, it returns row
numbers $$1 + in$$, where $$i = 0, 1, 2 ...$$)

*Arguments:*

1 required: A number `n` specifying the interval between rows.

*Examples:*

```
nfu words -E 3
```
