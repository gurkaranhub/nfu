*Description:*

Appends data repeated $$n$$ times.

*Arguments:*

2 required:
1. A pseudofile pattern.
2. A number `n` specifying the number of times the data is to be repeated.

*Notes*

Spencer: "This operator sucks."

*Examples:*

```
nfu --repeat 5 n:10
```
